Sobre o Desenvolvedor

Luiz Augusto, Técnico em Informática formado pelo SENAC. Apaixonado por desenvolvimento de software e criação de soluções que organizam a rotina.

O Meus Backlog foi desenvolvido como um projeto de portfólio para gerenciar coleções de entretenimento.

Tecnologias Utilizadas

  • Backend: Java com Spring Boot.
  • Frontend: HTML5, CSS3 e Thymeleaf.
  • Banco de Dados: PostgreSQL (Hospedado na Neon.tech).
  • Hospedagem: Render.com.

APIs e Recursos

  • TMDB: Fornecimento de posters e dados de filmes/séries.
  • Google Fonts: Fonte Poppins.
  • SweetAlert2: Alertas e diálogos interativos.
